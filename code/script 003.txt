=== KEY LEARNING POINTS ===
1. Oracle uses LISTAGG() instead of STRING_AGG()
2. Always include WITHIN GROUP (ORDER BY ...) clause
3. Use any delimiter: comma, pipe, semicolon, etc.
4. Combine with GROUP BY for powerful data summarization
5. Format data before aggregating for better readability
6. Use DISTINCT to avoid duplicate values in the list

Common uses: Creating reports, summary lists, CSV exports
